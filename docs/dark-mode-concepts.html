Shokunin Dark Mode Concepts

Static previews for a second VS Code theme choice. The syntax keeps Shokunin's current contract: sumi-like keywords/classes, asagi functions, ume type hints, tea strings, kintsugi constants, quiet properties, diagnostic-only red.

My top 3

  1. Yoru Sumi - closest dark translation of the current theme.
  2. Tsuki Ishi - most atmospheric, Kanagawa-adjacent without copying it.
  3. Kuro Kintsugi - warmest and most distinctive, but more opinionated.

1. Yoru Sumi

Warm charcoal workshop. Direct dark sibling for the rice-paper light theme: calm, readable, not black, not neon.
