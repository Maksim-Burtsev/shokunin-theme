# Palette

Final selected light variant: **Sumi / Ume Type**.

| Role | Color | Notes |
|---|---:|---|
| Paper | `#FCFAF2` | Editor, sidebar, terminal background |
| Panel paper | `#F6EFE2` | Tabs, activity bar, headers |
| Ink | `#303841` | Main text |
| Sumi keyword | `#3A3E44` | `from`, `import`, `class`, `def`, `return`, control flow |
| Asagi function | `#006FAE` | Functions and methods; main visual navigation |
| Sumi class | `#3A3E44` | Final v6 #23 class color; deliberately not blue |
| Ume type | `#8F4155` | Type hints: `str`, `float`, `Path`, `Iterable[TokenRole]` |
| Tea string | `#157A5B` | Strings |
| Kintsugi amber | `#9A6000` | Constants, enum members, numbers |
| Quiet property | `#4E5962` | Properties and attributes |
| Dry wood decorator | `#8A4A1C` | Decorators |
| Comment | `#777269` | Comments |
| Diagnostic red | `#C84053` | Errors only |
